GitHub. HW_2   —  https://github.com/Svetlana1Ganzhina/GitHub_part_2
1. На локальном репозитории сделать ветки для:  — git branch name
- Postman
- Jmeter
- CheckLists
- Bug Reports
- SQL
- Charles
- Mobile testing

2. Запушить все ветки на внешний репозиторий.   —  git push -u origin --all

3. В ветке Bag Reports сделать текстовый документ со структурой баг репорта   —   git checkout Bag Reports && vim bug_report.txt

4. Запушить структуру багрепорта на внешний репозиторий   —   git add . && git commit -m  «structure bag report»   —  git push origin Bug_Report

5. Вмержить ветку Bag Reports в Main   —  git checkout main && git merge Bug_Reports

6. Запушить main на внешний репозиторий   —  git push -u origin main

7. В ветке CheckLists набросать структуру чек листа   —  git checkout CheckLists && cat > structure_checklists.txt

8. Запушить структуру на внешний репозиторий  —  git add . && git commit -m «structure checklists   —  git push origin CheckLists

9. На внешнем репозитории сделать Pull Request ветки CheckLists в main  —  in my repository click «Pull Request» button , write «Add structure checklists» and click «Create pull request» 
button , on next link click «merge pull request» and «confirm merge» 

10. Синхронизировать Внешнюю и Локальную ветки Main   — git checkout main  — git pull
